Command Line Options
====================

The majority of controls for Meer are within the ``meer.yaml`` file.

.. option:: -d, --daemon 

   This option tells to fork Meer into the background

.. option:: -c, --config

   This option tells what configuration file to use.  By default Meer uses ``/usr/local/etc/meer.yaml``.

.. option:: -h, --help

   Meer help screen.

.. option:: -q, --quiet

   This option to tells Meer to not output to the console.  However, logs are still send to the /var/log/meer directory.
